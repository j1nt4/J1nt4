## 🎯 커리어 방향 & 목표

```text
시스템, 네트워크 & 보안   →   데브옵스   →           데브섹옵스        
        (기초)                 (자동화)        (파이프라인에 내재된 보안)
```

> 🎯 **목표**: *"Shift Left Security"* 원칙에 따라, 개발 단계부터 운영 배포까지 보안 검증을 자동화하여 보안을 데브옵스 라이프사이클의 기본 요소로 만드는 것입니다.
